## Simple Token Transfer Smart Contract

This project implements a basic token transfer system using Solidity.

### Tools Used
- Solidity
- Remix IDE (Remix VM)

### Description
A smart contract named `SimpleToken` was created with a total supply of 1000 tokens assigned to the deployer.  
A transfer function was implemented to send tokens between addresses.

### Steps Performed
1. Wrote the smart contract in Solidity.
2. Compiled successfully in Remix.
3. Deployed using Remix VM.
4. Verified initial balance of 1000 tokens.
5. Transferred 100 tokens to another account.
6. Verified updated balances (900 and 100).

### Result
Token transfer executed successfully.
